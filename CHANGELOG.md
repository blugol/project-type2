# 변경 이력 (Changelog)

## [1.0.0] - 2026-01-31

### ✨ 추가된 기능

#### 경제 일정 캘린더
- 월별 캘린더 뷰로 경제 지표 발표 일정 표시
- 일정 클릭 시 상세 정보 자동 표시
- 오늘 날짜의 첫 번째 일정 자동 선택
- "일정을 클릭하면 관련 종목을 볼 수 있습니다" 안내 문구 추가

#### 경제 지표 상세 정보
- 전월 실적, 시장 예상, 실제 발표 수치 표시
- 괴리율 자동 계산 및 색상 표시 (빨강/파랑)
- 경제 지표 기준치 가이드 시스템 구현
- 지표 제목 옆에 기준점 자동 표시 (예: "기준: 50.0")
- 지표별 해석 가이드 (수치 높음/낮음 의미)

#### 관련 종목 정보
- 시가총액 기준 자동 분류 시스템
  - 대형주: 2조원 이상
  - 중형주: 5천억~2조원
  - 소형주: 5천억원 미만
- 3단계 토글 방식 (그룹별 펼치기/접기)
- 종목 정보 표시: 종목명, 종목번호, 관련재료/업황/섹터
- 종목 카드 클릭으로 클립보드 복사 기능

#### UI/UX
- 다크모드/라이트모드 전체 지원
- 통일된 디자인 시스템
  - 라이트모드: Stone-400 톤의 차분한 회색 배경
  - 테두리: Gray-800로 통일
- 반응형 디자인 지원

#### 데이터 시스템
- `events.json`: 경제 일정 데이터
- `stocks2.json`: 종목 데이터 (시가총액 포함)
- `indicatorGuides.ts`: 경제 지표 기준치 정보
- 지원 지표: PMI, CPI, 비농업고용(NFP), 실업률, 원유재고, GDP
